1. Start using system

Start:
To start working using the system You need to belong to specific Project's group or groups.
You can check out existing projects and join some projects' groups or create your own project in "Project Section".

Create your own project:
To create new project go to 'Project Section' by clicking button on account main page or using option accessed via navbar under Projects.
There will be additional hints what to do next.
If u decided to create your own project and did it already, find your project using 'existing project'.
After going into the new created project start working on it by creating your first
project's group. Upon creating one, you can also select which users should be included.
This is not a mandatory element though.
After creating first project's group you will obtain project privileges allowing you to make modifications.
Also, your account main page will change to show specific info about groups that you created or joined already.
Remember that still you can join more groups or create ones.

Join to existing project:
To start working on some existing projects first go to project section and click on 'Existing projects' button.
After this a list of all currently existing projects will show up.
Find project that seems interesting for you and go to the project page.
This step is important because there You can find out if and how work is divided between
groups which are working on specific parts of the entire project.
To check out specific information about selected project's groups, simply click on those which seem to be interesting for You.
After clicking, You will access specific group profile page.
On the page You can learn more about group and also send request to join if You decide to do so.
After sending request, You will have to wait If the group will decide to accept or reject it.
You'll obtain privileges to work under the project in this specific group right after request acceptance.

Create your first task:
To create your first task just use button placed in navbar.
The operation is not allowed until You belong to any project's groups.

4. GitLab integration

How to integrate with GitLab?
Go to your GitLab project and next Settings->Integrations and then:
Add url https://testmyjirr.herokuapp.com/gitlab/push-event
Select "Push events" and uncheck "Enable SSL verification". Then click "Add webhook button".

How to make properly links ?

Add task id to your commit message e.g message="#FIX 1234 Title of task"; In task which has id = 1234 commit will be visible.

Email:
Email in Gitlab and in MyJiR must be the same.



5. Github integration

How to integrate with Github ?
Go to your github repository and next Settings->Webhooks and then:
Click button Add webhook
Add payload url : https://testmyjirr.herokuapp.com/github/push-event.
Select content type : application/json
How to make properly links ?
Add task id to your commit message e.g message="#FIX 1234 Title of task"; In task which has id = 1234 commit will be visible.

Email
Email in Github and in MyJiR must be the same.
